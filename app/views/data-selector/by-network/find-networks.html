Explore monitoring networks

Automatic Urban and Rural Network (AURN)

Data from the AURN is used for national air quality monitoring. It provides the highest quality air pollution data in the UK.

Start date: 1 July 1972

UK Urban NO2 Network (UUNN)

The UUNN measures concentrations of NO2 at urban traffic sites.

Start date: 1 January 2020

Locally managed automatic network

Most locally managed automatic monitoring stations are run by local authorities. They measure air quality in local pollution hotspots. They use a range of different monitoring equipment and produce data of varying quality.

Start date: 1 January 1986

Automatic Hydrocarbon Network

Automatic hourly measurements of speciated hydrocarbons.

Start date: 1 January 1993

Non-automatic Hydrocarbon Network

The Non-Automatic Hydrocarbon Network measures ambient benzene concentrations.

Start date: 1 January 1993

Heavy metals

The HM Network monitors the concentrations in air, and the deposition rates of a range of metallic elements at urban, industrial and rural sites.

Start date: 1 January 1980

Polycyclic Aromatic Hydrocarbons (PAH)

This network measures ambient concentrations of polycyclic aromatic hydrocarbons (PAH) in the atmosphere. PAHs are a group of persistent organic pollutant compounds that contain 2 or more benzene rings. They are generally produced through incomplete combustion or pyrolysis.

Start date: 1991

Toxic Organic Micro Pollutants (TOMPs) Network

This network measures ambient air concentrations for a range of persistent organic pollutants (POPs)

Start date: 1991

Black carbon

The Black Carbon Network monitors black carbon concentrations.

Start date: 1 September 2006

Particle concentrations and numbers network

This network monitors airborne particulate matter.

Start date: 1 January 1980

United Kingdom Eutrophying & Acidifying Network (UKEAP)

This network monitors long-term pollutant trends in rural and remote locations. It is made up of the National Ammonia Monitoring Network, Acid Gas and Aerosol Network, Rural NO2 Network and the Rural Mercury Network

Start date: 1 January 1980

UKEAP: Acid Gas and Aerosol Network (AGA Net)

This network provides monthly average concentrations of acid gases and inorganic aerosol composition.

Start date:1993

UKEAP: Rural NO2 Network

Nitrogen dioxide measurements in this sampling programme are used to map the background concentrations for Pollution Climate Mapping (PCM) and modelling.

Start date: 4 January 1993

UKEAP: National Ammonia Monitoring Network (NAMN)

NAMN provides monthly gaseous ammonia (NH3) concentrations.

Start date: 1 January 1980

UKEAP: MARGA

No information available

Start date: 17 February 2009

UKEAP: Rural Mercury Network

Mercury (Hg) measurements at high temporal resolution are carried out at 2 sites. At Auchencorth Moss, particulate bound Hg (PBM) in PM2.5, gaseous oxidised Hg (GOM) and total elemental Hg (GEM) are measured, while at the Chilbolton Observatory only the total gaseous elemental mercury (TGM) is measured.

Start date: 1 January 1980

Start date: 1 January 1977

This network stopped at the end of 2005 - replaced by the AURN and UUNN.
